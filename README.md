# FusionStore - Demo de eCommerce

**FusionStore** es una maqueta funcional de tienda online, desarrollada para demostrar habilidades en front-end. El proyecto simula un catálogo de productos, permite agregar artículos a un carrito (con contador) y presenta un diseño moderno y responsivo.

## Características

- Catálogo de productos renderizado dinámicamente desde JavaScript.
- Carrito de compras con contador interactivo.
- Diseño **responsivo** (mobile, tablet, desktop).
- Menú hamburguesa en dispositivos móviles.
- Formulario de newsletter funcional (demo).
- Efectos visuales suaves (hover, transiciones, sombras).

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 (Flexbox, Grid, variables CSS)
- JavaScript (ES6+)
- Font Awesome para iconos
- Google Fonts (Inter)
